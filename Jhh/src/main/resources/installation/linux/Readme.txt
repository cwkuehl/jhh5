Installation for program JHH6 (c) 2018 cwkuehl.de

Generell wird eine aktuelle Java-Installation vorausgesetzt.

Installation fuer Windows:
1) Speichern der Batch-Datei als #InstallUpdateJhh6.cmd in einem leeren Verzeichnis, z.B. D:\Haushalt\JHH6.
2) Starten von cmd.exe.
3) Wechseln in das Verzeichnis:
 D:
 cd \Haushalt\JHH6
4) Ausfuehren der Datei #InstallUpdateJhh6.cmd.
5) Beim naechsten Mal starten mit #JHH6.cmd.

Installation fuer Linux (getestet fuer Ubuntu 16.04):
1) Speichern der Batch-Datei als #InstallUpdateJhh6.sh in einem leeren Verzeichnis, z.B. /opt/Haushalt/JHH6.
2) Starten einer Terminal-Sitzung.
3) Wechseln in das Verzeichnis und Ausfuehrungsberechtigung setzen:
 cd /opt/Haushalt/JHH6
 chmod 755 #InstallUpdateJhh6.sh
4) Ausfuehren der Datei: ./#InstallUpdateJhh6.sh.
5) Beim naechsten Mal starten mit ./#JHH6.sh.
