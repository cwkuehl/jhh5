==== Formular {wp400-title}

Dieses Formular erlaubt das komfortable Erfassen, Kopieren, Ändern und Stornieren von Anlage-Buchungen.
Buchungen können Zahlungen, Rabatte, Zinsen und Anteilsänderungen beinhalten.
